# Car Price Prediction

A machine learning web application that predicts the selling price of used cars based on user-input features.

## 🚀 Features
- Comprehensive data exploration and cleaning
- Model training and evaluation
- Live deployed web interface
- Easy input form for car parameters

## 🛠️ Tech Stack
Python, Scikit-learn, Pandas, NumPy, Flask (backend), Render (deployment)
